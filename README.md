# Personal Finance Tracker

## Overview

This is a **small basic web project** built using HTML, CSS, and JavaScript.
The application allows users to add and view their expenses and automatically calculates the total spending.

The purpose of this project is to practice **basic web development concepts** such as DOM manipulation, user input handling, and simple UI design.

## Features

* Add an expense with description and amount
* Display a list of entered expenses
* Automatically calculate the total expense amount

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

personal-finance-tracker
├── index.html
├── style.css
└── script.js

## How to Run

1. Download or clone the repository.
2. Open `index.html` in any web browser.
3. Enter an expense description and amount.
4. Click **Add Expense** to see it added to the list and included in the total.
